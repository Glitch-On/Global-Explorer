# 🌍 Global Explorer

## 📌 Project Overview

Global Explorer is a web application that allows users to explore countries around the world and view important information such as population, region, and currency. It also includes a currency converter feature for real-time exchange rates.

## 🌐 APIs Used

* REST Countries API – for country data
* ExchangeRate API – for currency conversion

## ✨ Features

* 🔍 Search countries by name
* 🌍 Filter countries by region
* 📊 Sort countries by population
* 💱 Convert currency based on selected country
* ⭐ Save favorite countries (optional)

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Fetch API)

## ⚙️ How to Run

1. Clone the repository
2. Open index.html in browser
